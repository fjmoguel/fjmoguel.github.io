Assignment 4: Latent Semantic Analysis (LSA) Search Engine Webpage

This project implements an interactive Latent Semantic Analysis (LSA) search engine designed to retrieve and rank documents based on their similarity to a user’s query. The project utilizes the 20 Newsgroups dataset, a popular corpus for text mining and natural language processing tasks, and follows a structured approach to dimensionality reduction using Singular Value Decomposition (SVD). The core functionality involves transforming documents into a term-document matrix using TF-IDF (Term Frequency-Inverse Document Frequency) to capture the importance of words in documents. Then, LSA is applied through SVD to reduce the dimensionality of the document space, allowing for more efficient query matching. When a user submits a query, it is transformed and projected into the same reduced space. The search engine computes cosine similarity between the query and all documents, returning the top 5 most relevant documents ranked by similarity. Results are displayed on a webpage, complete with document metadata and a bar chart visualization of the similarity scores. The project is built with Flask for the web application and Plotly for dynamic chart visualization, and is fully integrated with a GitHub Actions workflow to ensure consistent and reliable builds.

Midterm Project: XGBoost Review Rating Prediction

In this project, I built a classification model to predict Amazon movie review star ratings using XGBoost. This model leverages engineered features like review sentiment and mean movie rating, with hyperparameter tuning and SMOTE oversampling to address imbalances. I achieved high accuracy in predicting 5-star reviews, while focusing on improving performance for the challenging middle-range ratings. Dimensionality reduction and optimized weighting schemes helped balance the model’s sensitivity across all star ratings.
