CS506 Final Project: Box Office Revenue Prediction

This project focuses on predicting movie revenues using advanced machine learning techniques and a user-friendly web application. We utilized features such as budget, runtime, release month, and interaction variables to build a robust XGBoost model. The web app, developed with Flask, provides revenue predictions, confidence intervals, and feature importance visualizations. The project involved data preprocessing, feature engineering, and hyperparameter tuning, achieving an R² score of 0.78 and RMSE of $101,991,097.67.

BA476 Final Project: Flight Delay Prediction

For this project, our team analyzed flight delays to predict arrival delays based on key factors like departure hour, flight distance, and weather patterns. Using a cleaned dataset from Kaggle, we engineered predictive features and trained models such as Random Forest, Lasso, and Neural Networks. We implemented advanced techniques like cross-validation and feature transformation to reduce MSE, achieving insights into delay patterns at major U.S. airports.

CS411 Project: Software Engineering

This project focused on developing a Twitter API application to scan user profiles, track followers, and analyze event interests using AI matching algorithms for optimized results. The project used OAuth for authentication, implemented with Node.js and Passport. API calls to Twitter and Ticketmaster were made using Postman, and I developed both the back end and front end using Python, HTML, and CSS. This included integrating data into Google Maps to display local events. The project involved full-stack development, allowing users to find events based on their interests.
